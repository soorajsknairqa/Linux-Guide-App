🚀 Linux Pocket Guide – Interactive Learning

An interactive web-based Linux learning platform designed for both beginners and professionals. This project provides a structured, visually engaging way to learn Linux concepts, commands, and system operations.

📌 Features
🎯 Beginner & Professional Modes
Toggle between simple explanations and advanced system-level concepts
📚 Structured Learning Sections
Basics
Filesystem
Commands
Processes
Users & System Management
🔍 Real-time Search
Quickly find commands and topics
📋 Copy-to-Clipboard
Easily copy command examples
💻 Interactive UI
Clean design with responsive layout
Smooth animations and transitions
📱 Mobile Friendly
Fully responsive for all devices
