🚀 Getting Started with Linux

Linux is a powerful, open-source operating system. Here's what you need to know:

Beginner

🎯
What is Linux?

Linux is an open-source operating system kernel created by Linus Torvalds. It powers everything from servers to smartphones.

Linux is the core of the operating system. A "distribution" (or distro) combines Linux with other software to create a complete OS.
Beginner

💻
Shell Interface

The shell is your command-line interface to Linux. It interprets commands and communicates with the operating system.

Popular shells: Bash (most common), Zsh, Fish, and Ksh.
Beginner

📦
Distributions

Different Linux distros bundle the kernel with various tools. Ubuntu, Fedora, Debian, and CentOS are popular choices.

Check your distro
cat /etc/os-release
Professional

🔧
Kernel & Modules

The Linux kernel is the core managing hardware resources, memory, processes, and device drivers through loadable kernel modules.

Inspect kernel & modules
uname -r # Kernel version lsmod # List loaded modules modinfo module_name # Module details cat /proc/version # Detailed version info
Professional

🗂️
Init Systems

Init systems (systemd, sysVinit, OpenRC) manage services and boot sequence. systemd is the modern standard on most distributions.

Systemd commands
systemctl status service systemctl start|stop|restart service systemctl enable service # Enable on boot journalctl -u service # View service logs
Professional

📊
System Architecture

Understanding system architecture including CPU, memory hierarchy, I/O subsystems, and kernel space vs. user space interactions.

Architecture inspection
lscpu # CPU architecture dmidecode # Hardware info cat /proc/cpuinfo # CPU details cat /proc/meminfo # Memory details

📁 Filesystem & Directory Structure

Understanding the Linux directory hierarchy is essential for navigation.

Beginner

🌳
Root Directory (/)

The top-level directory containing all files and folders on your system.

Key directories:
/bin - Essential commands
/etc - Configuration files
/home - User directories
/usr - User programs
/var - Variable data
Beginner

🏠
Home Directory

Your personal directory where you store files. Represented by the tilde (~) symbol.

Navigate to home
cd ~ pwd # Print working directory
Beginner

🔍
Path Navigation

Paths can be absolute (from root) or relative (from current directory).

Path examples
# Absolute path /home/username/documents # Relative path ./documents ../parent_folder
Professional

🔗
Inode & Hard Links

Inodes store metadata for files. Hard links reference the same inode while symbolic links point to file paths.

Inode operations
ls -i # Show inode numbers ln file hardlink # Create hard link ln -s file symlink # Create soft link stat file # Detailed inode info find / -inum 12345 # Find by inode
Professional

📦
Filesystem Types

Linux supports multiple filesystem types (ext4, btrfs, XFS, NFS). Each optimized for different use cases and performance characteristics.

Filesystem inspection
df -T # Show filesystem types mount # Mounted filesystems tune2fs -l device # ext4 details blkid # Block device UUIDs lsblk # Block device tree
Professional

⚡
File Descriptors & Pipes

File descriptors (0-2 for stdin/stdout/stderr) and pipes enable powerful command chaining and I/O redirection strategies.

Advanced I/O redirection
cmd 2>&1 | tee output.log # Capture all cmd > /dev/null 2>&1 # Discard output exec 3< file; read var <&3 # File descriptor mkfifo pipe; cat pipe & # Named pipe ls | tee file1 file2 # Multi redirect

⚡ Essential Commands

Master these fundamental commands for daily Linux work:

Beginner
ls
List files and directories
ls -la

Show all files with details in list format

Pro Tips: ls --color=auto, ls -lh (human readable), ls -S (sort by size)
Beginner
cd
Change directory
cd /path/to/directory

Navigate to a different directory

Pro Tips: cd - (previous dir), cd without args (home), CDPATH variable
Beginner
cat
Display file contents
cat filename.txt

View entire file content on screen

Pro Tips: cat -n (line numbers), cat -A (show all chars), cat file1 file2 > combined
Beginner
cp
Copy files and directories
cp -r source/ dest/

Copy with -r for recursive directory copy

Pro Tips: cp -P (preserve links), cp -a (archive mode), cp -u (update only)
Beginner
mv
Move or rename files
mv oldname newname

Rename or move files between directories

Pro Tips: mv -i (interactive), mv -v (verbose), atomic operation on same filesystem
Beginner
rm
Remove files and directories
rm -rf directory/

⚠️ Use carefully! -rf removes recursively

Pro Tips: rm -i (confirm), rm -d (empty dirs), shred (secure delete)
Beginner
grep
Search text patterns
grep "pattern" file.txt

Find lines matching a pattern

Pro Tips: grep -E (extended regex), grep -r (recursive), grep -c (count)
Beginner
find
Search for files
find . -name "*.txt"

Find files matching criteria recursively

Pro Tips: find -exec, find -type f|d, find with predicates, xargs integration
Beginner
chmod
Change file permissions
chmod 755 script.sh

Set read, write, execute permissions

Pro Tips: chmod u+x (symbolic), chmod -R (recursive), umask, ACLs (setfacl)
Beginner
mkdir
Create directories
mkdir -p parent/child/

-p creates parent directories too

Pro Tips: mkdir -m (set perms), mkdir with mode 777, shell brace expansion
Beginner
tar
Archive files
tar -czf archive.tar.gz folder/

Create compressed archive

Pro Tips: tar -j (bzip2), tar -x (extract), tar -C (change dir), incremental backups
Beginner
ssh
Secure shell access
ssh user@remote.host

Connect to remote server securely

Pro Tips: ssh-keygen, ssh config file, ssh tunneling (-L/-R), scp for files
Professional
awk
Pattern scanning and processing language
awk '{print $1, $NF}' file

Powerful text processing with fields, regex, and programming constructs

Professional
sed
Stream editor for filtering/transforming
sed 's/old/new/g' file

Non-interactive editor for sed scripting and complex replacements

Professional
strace
Trace system calls and signals
strace -e trace=file ls

Debug applications by tracing syscalls, performance analysis

Professional
ltrace
Trace library function calls
ltrace ./program

Trace dynamic library calls for deeper debugging

Professional
ldd
Print shared library dependencies
ldd /usr/bin/ls

Analyze linked libraries and potential dependency issues

Professional
iproute2
Advanced network configuration (ip command)
ip addr show; ip route show

Modern replacement for ifconfig/route with advanced options

💡 Tip: Use man command to view detailed documentation for any command.

⚙️ Process Management

Monitor and control running programs on your system:

Beginner

👁️
Viewing Processes

See what's running on your system with these commands:

View all processes
ps aux # All processes with details top # Real-time process monitor htop # Interactive process viewer
Beginner

🛑
Stopping Processes

Terminate running processes with kill signals:

Kill a process
kill PID # Graceful termination kill -9 PID # Force kill killall process_name # Kill by name
Beginner

🔄
Background Jobs

Run processes in the background while using the terminal:

Background operations
command & # Run in background Ctrl+Z # Suspend current process bg # Resume in background fg # Bring to foreground jobs # List background jobs
Professional

📊
Process Signals & State

Understanding Linux signals and process states for advanced process control:

Signal handling
kill -l # List all signals kill -SIGTERM pid # Graceful shutdown kill -SIGKILL pid # Force kill (cannot ignore) trap 'code' SIGTERM # Shell signal handler ps aux | grep state # View process states
Professional

⚡
CPU & Memory Profiling

Deep dive into process resource usage and performance bottlenecks:

Advanced monitoring
ps -eo pid,comm,%cpu,%mem --sort=-%mem pmap -d pid # Memory mapping details /proc/[pid]/status # Detailed process stats perf record -p pid # CPU profiling valgrind --leak-check ./program # Memory leaks
Professional

🔀
Process Relationships

Understanding parent-child relationships, process groups, and sessions:

Process tree & relationships
pstree -p # Process tree with PIDs ps -eo pid,ppid,cmd # Parent-child view ps -eo pid,pgid,sid # Process group and session ps j # Job control info /proc/[pid]/cwd # Working directory

Sudo usage

System Information

Get details about your system and hardware:

System info

👥
User & Group Administration

Create, modify, and manage user accounts and groups with advanced options:

User management
useradd -m -s /bin/bash user # Create user usermod -aG group user # Add to group userdel -r user # Delete with home groupadd groupname # Create group /etc/passwd, /etc/shadow # User database files
Professional

🔑
PAM & Authentication

Pluggable Authentication Modules for advanced authentication mechanisms:

PAM configuration
/etc/pam.d/ # PAM configuration passwd -S user # Check password status passwd -e user # Expire password shadow -l # Show shadow file format getent passwd user # Query user database
Professional

🛡️
SELinux & AppArmor

Mandatory Access Control systems for security enforcement at kernel level:

Security contexts
getenforce # Check SELinux mode ls -Z # Show SELinux context semanage fcontext -l # List contexts aa-status # AppArmor status aa-audit # Audit mode
Professional

🔐
Sudo Configuration

Configure sudoers file for fine-grained privilege delegation and auditing:

Sudoers management
visudo # Safely edit sudoers sudo -l # List permissions sudo -U user -l # Check other user sudoedit /etc/file # Safe file editing /var/log/auth.log # Sudo audit log
